Posted on 2021-02-04 In 技术 Views:
  1. 安装NexT： hexo 5 支持用npm安装主题了
    npm install hexo-theme-next
    会安装到\node_modules\hexo-theme-next中

换theme后，最好hexo clean 清理cache first.
以调试方式运行。Hexo local server with debug parameter (by adding –debug)：
hexo s –debug

  1. 安装Utterances：用readme指导更新：
    一定要在github先建立一个新的repo
    repository: chaihuaa/blogcomment

然后在github的setting中安装配置application： utterances
然后在本地目录安装

npm install theme-next/hexo-next-utteranc

激活靠改\node_modules\hexo-theme-next_config.yml的utterances部分，只需要改两行：
>enable: true
>repo: chaihuaa/blogcomment #Warehouse name, format: your user ID/warehouse name
nexT会自动在页面尾部位置加入comment部分.

https://www.npmjs.com/package/hydai-hexo-next-utteranc
You can config those in both hexo or theme _config.yml
